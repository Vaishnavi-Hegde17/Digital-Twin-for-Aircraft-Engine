Engine Overview

Model: Adour Mk-821
A turbofan engine used in various military aircraft platforms.

About this Digital Twin

This Digital Twin replicates the HAL Adour turbofan engine behaviour using physics-inspired synthetic data and machine learning. It supports:

  • Real-time sensor streaming and visualization
  • Health classification: NORMAL / WARNING / CRITICAL
  • Predictive maintenance using RandomForest, XGBoost, and Neural Network models
  • Email alerts on detected anomalies

Engine Sensors & Normal Operating Ranges

RPM (Revolutions per Minute)

Normal Range: 2,000 - 8,500 RPM
Critical if above 9,000 or below 1,500

EGT - Exhaust Gas Temperature

Normal Range: 500°C - 800°C
Critical if above 850°C or below 400°C

Fuel Flow

Normal Range: 1,500 - 4,500 kg/h
Critical if above 5,000 or below 1,000 kg/h

Vibration

Normal Range: 0 - 5.0 mm/s
Critical if above 8.0 mm/s

Oil Temperature

Normal Range: 60°C - 110°C
Critical if above 120°C or below 50°C

Oil Pressure

Normal Range: 3.0 - 7.0 bar
Critical if above 8.0 or below 2.5 bar

How to use

  1. Register & login via the left sidebar menu
  2. Open Dashboard to view live telemetry with real-time data streams
  3. Use Prediction page to test custom input scenarios and get health predictions
  4. Monitor sensor values against normal ranges to identify anomalies

Model Notes

Models are trained on realistic synthetic dataset and saved under Backend/savedmodels. Live predictions use the RandomForest model by default; you can configure models in Backend/config.json.
